## Overview

This project is a web-based Finite Automata Simulator designed to construct, visualize, and simulate both Deterministic Finite Automata (DFA) and Non-Deterministic Finite Automata (NFA).

The simulator provides an interactive interface where users can define states, transitions, and input strings, and observe how the automaton processes the input. It also includes basic theoretical explanations to support learning.

---

## Objectives

- To implement DFA and NFA simulation in an interactive environment  
- To provide visualization for better understanding of automata behavior  
- To reinforce theoretical concepts from Automata Theory through practical implementation  

---

## Features

- Creation and deletion of states  
- Marking of start and accepting states  
- Addition of transitions with symbols (including ε for NFA)  
- Toggle between DFA and NFA modes  
- Step-by-step and complete simulation of input strings  
- Display of acceptance or rejection results  
- Integrated theory section for reference

## Methodology

The system allows users to define an automaton by creating states and transitions.

- In DFA, each state has exactly one transition per input symbol  
- In NFA, multiple transitions and ε-transitions are allowed  

The simulator processes the input string either step-by-step or in a single run and determines whether the string is accepted or rejected based on the defined automaton.
